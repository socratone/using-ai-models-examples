## git lfs 설치

hugging face의 모델을 다운 받기 위해서 git을 이용할 수 있는데,\
git을 이용해서 대용량 파일을 다운 받으려면 git lfs를 설치해야 한다.

맥에서는 brew를 이용해서 설치한다.

```
brew install git-lfs
```

lfs를 사용할 수 있도록 git 업데이트

```
git lfs install
```
